## 1. 產品願景

打造一個以「專業交易者工作區（Trader Workspace）」為核心設計語言的股票平台，三大對標軸線：

| 維度 | 對標 | 我們的超越點 |
|------|------|-------------|
| 圖表深度 | TradingView | 整合台股三大法人籌碼疊圖，原生支援台股技術指標 |
| 綜合體驗 | 富途牛牛 | 更輕量、更快速、無強迫開戶流程，開放 API 整合 |
| 籌碼分析 | 籌碼K線 | 加入 AI 策略選股，並提供可程式化的策略回測框架 |

**核心主張（Positioning）：** 「台股最聰明的工作空間——從籌碼到 AI，一個螢幕全看透。」

---

## 2. 目標用戶

### 主要用戶（Primary）
- **主力交易者（Active Trader）：** 每日盤中盯盤，需要多視窗、快速切換、即時籌碼
- **量化研究者：** 需要乾淨資料 API、策略回測、可腳本化的指標

### 次要用戶（Secondary）
- **長線投資人：** 偶爾查看基本面摘要 + 月線籌碼趨勢
- **財經媒體 / 部落客：** 使用嵌入圖表功能

- **主題：** 深色模式（Dark Mode）預設，支援淺色切換
- **字型：** 數字使用等寬字型（JetBrains Mono / Roboto Mono），中文 Noto Sans TC

### 4.3 各面板資訊流向說明

**LEFT PANEL（導航層）**
- 資訊流：由用戶驅動（主動搜尋、點選自選股）→ 觸發 CENTER 主圖切換
- 功能：自選股群組切換、市場熱門即時排行、產業板塊地圖入口、最近瀏覽記錄
- 可折疊：按鈕折疊成 icon-only 模式，釋放主圖空間

**CENTER PANEL（分析核心）**
- 資訊流：接收左側股票選擇信號 → 載入 K 線數據 + 指標疊加 → 底部標籤切換不同分析維度
- 底部標籤內容：
  - `圖表` — 多時框 K 線主圖（預設啟動頁）
  - `基本面` — 財務概覽（EPS、毛利率、收入趨勢）
  - `新聞` — 個股相關財經新聞（整合 RSS / 爬蟲）
  - `籌碼詳情` — 三大法人 30 日詳細表格 + 融資券
  - `財報` — 季報 / 年報 PDF 連結 + 重點摘要
- 佈局切換：右上角支援 1 圖 / 2 圖 / 4 圖多開模式

**RIGHT PANEL（情報層）**
- 資訊流：跟隨 CENTER 當前選定股票 → 自動更新個股摘要卡
- 設計哲學：提供「決策所需的最少關鍵資訊」，避免資訊噪音
- 可切換為「螢幕監控模式」：關閉右側面板，進入極簡盯盤視圖

### 4.4 行動裝置適配（RWD）

- `≥ 1280px`：三欄全展開工作區
- `768px – 1279px`：LEFT PANEL 折疊，兩欄佈局（清單 + 圖表）
- `< 768px`：單欄，底部 Tab Bar 切換「行情 / 圖表 / 選股 / 自選」

## 5. 技術棧推薦

### 5.1 Frontend

| 技術 | 選擇 | 理由 |
|------|------|------|
| 框架 | **Next.js 15**（App Router） | SSR/SSG 兼容、SEO 友好、React 生態 |
| 語言 | **TypeScript 5.x** | 金融數據型別複雜，強型別必要 |
| 圖表庫 | **TradingView Lightweight Charts v5** | 效能最佳、台股 K 線原生支援 |
| 副圖指標 | **Apache ECharts** | 處理法人疊圖、熱力圖靈活度更高 |
| 狀態管理 | **Zustand** | 輕量、支援 WebSocket 訂閱流 |
| 樣式 | **Tailwind CSS v4** + CSS Variables | 主題切換（Dark/Light）最省力 |
| 拖曳佈局 | **dnd-kit** | 自選股排序 + 面板大小調整 |
| 表格 | **TanStack Table v8** | 大量行情數據排序/篩選效能優 |
| 表單 | **React Hook Form + Zod** | 策略條件輸入驗證 |

### 5.2 Backend

| 技術 | 選擇 | 理由 |
|------|------|------|
| 框架 | **FastAPI（Python）** | 量化計算生態豐富、非同步效能好 |
| 即時推播 | **WebSocket（FastAPI WebSocket）** | 行情 tick 推播 |
| 任務排程 | **APScheduler** | 盤後資料抓取排程，內建於 FastAPI 程序，免額外服務 |
| 資料庫 | **Supabase（PostgreSQL）** | 免費 500MB + 內建 Auth + Row-level Security |
| 快取 | **Upstash Redis** | 免費 tier（10,000 命令/天），Serverless 計費 |
| 搜尋 | **Meilisearch**（自架於 Fly.io）| 免費自架，股票代碼模糊搜尋 |
| AI 整合 | **Google Gemini 1.5 Flash API** | 免費 tier 1,500 次/日，自然語言選股解析 + 摘要 |
| 佇列 | **Upstash Redis Streams** | 行情資料管線，同一免費帳戶 |
