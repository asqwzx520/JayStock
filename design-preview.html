04 / 升級對照表
項目 現況 (V1) 建議升級 (V2) 影響 說明
品牌色 #3B82F6 普通藍 #00D4FF 電光青藍 高 區別於大多數平台的 Tailwind 藍，有獨特金融科技感
背景層次 #0a0e17 / #0d1220 2層 5層深度系統 高 卡片、側欄、Popover 有明確的視覺深度，不再平扁
邊框 #1a2333 灰色 rgba(0,212,255,0.12) 青藍半透明 中 邊框帶品牌色調，整體更有系統感
主字型 Noto Sans TC / Inter Space Grotesk + Syne (display) 中 Space Grotesk 數字更緊湊清晰；Syne 用於 Logo/標題更具個性
數字字型 JetBrains Mono IBM Plex Mono 低 IBM Plex Mono 在小字號更清晰，數字字寬更緊湊
全螢幕按鈕 藏在可捲工具列裡 固定在工具列最右側 高 ✅ 已修復（本 session 完成）
Glow 效果 基本 text-shadow 三層 glow + 品牌色 border 中 活躍按鈕加 box-shadow glow，hover 時更有互動感
Header 指數 只有加權指數 加權 + NYSE + USD/TWD + 即時點 中 提供更豐富的市場 context，符合專業看盤習慣
Screener 表格 簡單 tr hover 評分條 + 入場動畫 + 自選股一鍵 高 評分視覺化更直觀；結果載入有 staggered 動畫；+ 按鈕永遠可見
噪點紋理 無 body::before SVG noise 1.8% opacity 低 極輕微的紋理讓純黑背景不顯呆板，增加質感
05 / globals.css 可直接替換的 Token
/* ── 品牌色升級 ── */ --color-brand: #00D4FF; /* 電光青（取代 #3B82F6）*/ --color-brand-hover: #00B8E6; --color-up: #FF4757; /* 更飽和的漲紅 */ --color-down: #2ED573; /* 更亮的跌綠 */ /* ── 背景層次升級（深色主題）── */ --bg-base: #070C18; --bg-surface: #0C1628; --bg-elevated: #111E35; /* ── 邊框改為品牌色調 ── */ --border: rgba(0, 212, 255, 0.12); --border-strong: rgba(0, 212, 255, 0.22); /* ── 文字色調整（冷白）── */ --text-primary: #E8F4FF; --text-secondary: #7A9CC8; --text-tertiary: #3D5A80; /* ── 字型升級 ── */ --font-sans: "Space Grotesk", "Noto Sans TC", system-ui, sans-serif; --font-mono: "IBM Plex Mono", "JetBrains Mono", monospace;
字型載入（layout.tsx 加入）：
<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700&family=Syne:wght@700;800&family=IBM+Plex+Mono:wght@400;500;600&display=swap" rel="stylesheet">

或改用 next/font（推薦，避免 FOUC）：
import { Space_Grotesk, IBM_Plex_Mono, Syne } from 'next/font/google'
